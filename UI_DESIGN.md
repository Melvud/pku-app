## Color Scheme

### Admin Panel Colors
- **Primary**: Purple/Indigo gradient
- **Statistics Cards**: 
  - Blue (Users)
  - Green (Active Users)
  - Orange (Recipes)
  - Teal (Approved)
  - Amber (Pending)
  - Purple (Activity)

### Action Colors
- **Approve**: Green (#4CAF50)
- **Reject**: Red (#F44336)
- **Neutral**: Grey (#757575)
- **Info**: Blue (#2196F3)

### State Indicators
- **Pending**: Amber badge with clock icon
- **Approved**: Green badge with checkmark
- **Rejected**: Red badge with X icon
